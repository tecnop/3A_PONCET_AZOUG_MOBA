Rappels:
La barre espace permet de déverouiller la caméra, qui se déplace ensuite en approchant la souris des bords de l'écran
Les touches ZQSD (par défaut) permettent de se déplacer
En cliquant sur la partie inférieure gauche de l'interface, vous pouvez accéder à une vue simplifiée des stats
Les cylindres bleus sont des objets; cliquer dessus ouvre une fenêtre permettant de les équiper ou de les recycler. Recycler une arme permet d'obtenir de l'expérience, utilisée pour déverrouiller des compétences.

Lors de l'arrivée au menu, il est possible de choisir entre les modes "suprématie" et "course à la gloire".
Dans le mode "Cours à la Gloire", le but est de vaincre l'ennemi se trouvant en haut de la carte, récupérant ainsi une relique, qu'il faut alors amener à son camp pour gagner. Tuer le joueur possédant la relique détruit la relique et fait réapparaitre l'ennemi. En "Suprématie", il faut tuer le seigneur ennemi.

Notez aussi qu'il est possible d'activer ou de désactiver le mode "Sécurisé". Désactiver ce mode indique au jeu de lire un fichier de propriété externe pour initialiser les tables de données. Joué en ligne, ce mode fera télécharger aux clients la configuration du serveur.

En bas de l'interface se trouve une barre de sorts, qui affiche les sorts actuellement connus par le joueur. Cliquer sur un des emplacements permet de changer le sort qui y est équipé (le sort par défaut est une attaque simple). Les touches pour activer ces sorts sont, dans l'ordre: clic gauche, clic droit, clic milieu, 1, 2, 3, 4, R

Notez qu'un des sorts (Tir multiple) nécessite qu'un arc soit équipé

Comme précédement, des armes de tests seront présentes près du point d'apparition du joueur 1, ainsi que des monstres déposant une panoplie très puissante.

La partie contre une IA n'est pas encore tout à fait opérationelle, mais le système est là.
